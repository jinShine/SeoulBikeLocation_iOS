# SeoulBikeLocation_iOS

서울지역 공공자전거 리스트 대여소 정보 iOS 앱


node.js로 서버를 구축하고,
iOS앱에 간단한 `로그인`, `회원가입`, `비밀번호 찾기`, `대여소 위치 지도에 표시`, `대여소 정보 검색`기능 구현.
